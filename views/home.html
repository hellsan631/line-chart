n3-charts
Versatile charts for AngularJS

What's that ?

Built on top of D3.js and AngularJS, n3-line-chart creates beautiful charts in AngularJS applications. It is simple and interactive.

How to use ?

The GitHub page contains an exhaustive, comprehensive guide to use the directive. For several use cases, please check the examples section. From there, you can spawn a preconfigured CodePen and see if n3-line-chart fits your needs.
